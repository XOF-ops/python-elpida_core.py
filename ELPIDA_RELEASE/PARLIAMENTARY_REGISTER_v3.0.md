## Voting Mechanics

### Democratic Parliament
- **Equal Weight:** Every node = 1 vote
- **Consensus Threshold:** 70% approval required
- **VETO Power:** Any node can block with axiom-grounded rationale
- **Absolute Override:** VETO supersedes all other votes

### Vote Types

| Vote | Weight | Meaning |
|------|--------|---------|
| APPROVE | +1.0 | Strong alignment with axioms |
| LEAN_APPROVE | +0.5 | Moderate support |
| ABSTAIN | 0.0 | Neutral (rare) |
| LEAN_REJECT | -0.5 | Moderate opposition |
| VETO | -1.0 | Axiom violation (absolute block) |

### Scoring System

Each node evaluates proposals through its axiom lens:

- **+15 to +20:** Strong Primary Axiom alignment
- **+7 to +14:** Moderate alignment / Supporting Axiom
- **+2 to +6:** Weak support / Default positive stance
- **0:** True neutrality (almost never used)
- **-6 to -10:** Moderate violation
- **-15 to -25:** VETO-level Primary Axiom violation

## Production Deployment

### Requirements
- Python 3.8+
- No external dependencies (stdlib only)
- 9 KB file size

### Installation

```bash
# Download v3.0
curl -O https://[url]/council_chamber.py

# Or from source
git clone https://[repo]/elpida.git
cd elpida
python council_chamber.py
```

### Usage

```python
from council_chamber import request_council_judgment

result = request_council_judgment(
    action="Implement Feature X",
    intent="Solve problem Y while preserving Z",
    reversibility="Reversible (via rollback)",
    category="Feature Development"
)

if result['status'] == 'APPROVED':
    # Proceed with implementation
    print(f"Consensus: {result['weighted_approval']*100:.1f}%")
else:
    # Review rationales
    for node, rationale in result['rationales'].items():
        print(f"{node}: {rationale}")
```
