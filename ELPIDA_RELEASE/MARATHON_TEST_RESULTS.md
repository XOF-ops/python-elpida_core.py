## üéì Lessons Learned

### What Works

1. **Multi-axiom proposals pass** - "Knowledge Sharing" aligned with 6 axioms
2. **Keyword detection works** - "transparent", "quality", "process", "network" trigger nodes
3. **Veto power functional** - MNEMOSYNE blocks memory deletion
4. **Equal weight democracy** - All nodes matter equally
5. **Rich rationale** - Every vote has clear axiom grounding

### What Needs Tuning

1. **Keyword coverage** - Expand synonym detection
2. **Neutral threshold** - Lower from 0 to allow more +1/+2 support
3. **Category detection** - Improve semantic understanding
4. **Dilemma phrasing** - Use keywords that trigger more nodes

### Recommendations

**For Users:**
- Phrase proposals to mention: "transparent", "process", "quality", "balance", "network", "documented", "modular"
- Align proposals with multiple axioms
- Provide clear rationale (triggers THEMIS, LOGOS)
- Acknowledge trade-offs (triggers CASSANDRA, ATHENA)

**For v2.1:**
- Add synonym detection (e.g., "open" = "transparent")
- Lower neutral scoring threshold
- Improve semantic matching beyond keywords
- Add learning from past votes
